# ClubFlow Admin Web

Starter Next.js scaffold for the separate admin panel.

## Purpose

This app is intended for club staff to manage:

- trainings
- trainers
- customers
- settings

The Android app should remain customer-focused.

## Current state

This scaffold currently includes:

- dashboard page
- trainings page
- trainers page
- customers page
- settings page
- shared mock data and types

## Next steps

1. Install dependencies with `npm install`
2. Run locally with `npm run dev`
3. Replace mock data with real API calls
4. Add authentication and role-based access
5. Add create/edit mutations
